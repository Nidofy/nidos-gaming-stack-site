# game-content-author

## name

game-content-author

## description

为站点撰写、整理或校验游戏内容的 skill，覆盖中英文游戏详情、摘要、卖点、FAQ、媒体说明和 SEO 字段。

## 何时触发

- 用户要求新增游戏内容。
- 用户要求改写中文或英文项目介绍。
- 用户要求为公众号、媒体或下载页准备文案。
- 用户要求检查中英文内容一致性。

## 工作步骤

1. 读取 `AGENTS.md`、`spec/CONTENT_MODEL.md`、`spec/I18N_SPEC.md` 和 `spec/SEO_SPEC.md`。
2. 收集游戏事实：玩法、平台、状态、版本、截图、限制、下载方式。
3. 按内容模型补齐中英文字段。
4. 中文文案偏传播和故事性，英文文案偏国际玩家和媒体理解。
5. 检查是否有夸大、不确定日期、未经授权商标或误导性表述。
6. 输出内容文件或文案差异说明。

## 输入材料

- 游戏介绍草稿。
- 当前开发状态。
- 版本和下载信息。
- 截图、视频、FAQ、已知问题。

## 输出文件

- `src/content/games/*.mdx`
- `src/content/posts/*.mdx`
- SEO 字段或分享摘要。
- 内容审校建议。

## 验收标准

- 中文和英文结构一致。
- 文案事实准确，未承诺未完成能力。
- 字段满足 schema。
- 有合适的 SEO title、description 和社交分享摘要。
- 授权和商标边界清晰。

## 禁止事项

- 不直接发布未经确认的机器翻译。
- 不写虚假发布日期、平台支持或媒体评价。
- 不堆砌 Dota、Valve、The International 等商标关键词。
- 不把公众号风格文案原封不动塞进英文页面。
