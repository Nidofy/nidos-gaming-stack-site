# wechat-campaign-writer

## name

wechat-campaign-writer

## description

为微信公众号宣传和中文落地页撰写内容的 skill，帮助中文读者从文章进入站点、了解项目并下载或关注更新。

## 何时触发

- 用户要求写公众号文章。
- 用户要求创建或修改微信落地页。
- 用户要求把游戏更新包装为中文传播内容。
- 用户要求生成微信分享标题、摘要和封面建议。

## 工作步骤

1. 读取 `AGENTS.md` 和 `spec/WECHAT_PROMOTION_SPEC.md`。
2. 确认 campaign 关联的 game slug 和目标行为。
3. 提炼公众号文章 hook、项目上下文、读者收益和 CTA。
4. 输出落地页内容结构，并链接正式游戏详情页和下载页。
5. 检查移动端阅读、微信内置浏览器、分享摘要和素材授权。

## 输入材料

- 游戏当前状态。
- 本次传播主题。
- 公众号文章草稿或要点。
- 截图、视频、下载入口和 FAQ。

## 输出文件

- `src/content/wechat/*.mdx`
- 微信分享标题、摘要、封面建议。
- 公众号文章大纲或正文草案。

## 验收标准

- 落地页与正式游戏页互相链接。
- 中文表达适合传播但不过度承诺。
- CTA 明确。
- 移动端阅读无明显障碍。
- 未使用未经授权的真实赛事或商标视觉。

## 禁止事项

- 不把微信落地页当成唯一正式介绍页。
- 不写夸张营销承诺。
- 不制造与第三方赛事、公司或组织的关联暗示。
- 不使用只能在微信内理解的上下文作为页面核心信息。
